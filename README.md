# 🏥 Hospital Emergency Management Dashboard

## 📌 Overview
This project is an interactive Hospital Emergency Management Dashboard developed in Microsoft Excel. It helps hospitals monitor emergency operations, patient flow, bed availability, doctor assignments, and key performance metrics for better decision-making.

## 🚀 Features
- Emergency Patient Dashboard
- Patient Admission & Discharge Tracking
- Doctor Performance Analysis
- Bed Occupancy Monitoring
- Department-wise Analysis
- Interactive Slicers & Filters
- KPI Cards and Charts

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Power Query
- Data Cleaning
- Dashboard Design

## 📂 Files Included
- Excel Dashboard (.xlsx)
- Dataset (.csv)
- Dashboard Screenshot (.png)
- Project Presentation (.pptx)
